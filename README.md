# Mary&May TikTok 어필리에잇 대시보드

TikTok Shop US 어필리에잇 캠페인 진행 현황을 크리에이터/영상 단위로 보여주는 웹 대시보드입니다.
구글시트 **"어필리에잇 로우데이터"**를 원본으로 하며, 페이지를 열 때마다 최신 데이터를 자동으로 불러옵니다. (수동 업로드 불필요)

## 화면 구성

- **KPI 요약**: 참여 크리에이터 수, 전체 업로드 영상 수, 누적 GMV, GMV MAX/스파크애즈 광고비
- **크리에이터 카드 (요구사항 1)**: 유저네임 옆에 티어를 직접 입력/관리, 영상 링크를 업로드 일시 최신순으로 표시
- **영상별 광고 성과 (요구사항 2)**: 영상을 펼치면 GMV MAX 노출/클릭/CTR/광고비를 영상 단위로 정확히 매칭해서 보여주고, 스파크애즈는 크리에이터 단위 합산으로 카드 상단에 표시

## 데이터 연결 방식 (중요 — 반드시 읽어주세요)

이 대시보드는 구글시트의 `gviz` 공개 조회 API를 이용해 브라우저에서 직접 데이터를 가져옵니다.
그래서 **구글시트가 반드시 "링크가 있는 모든 사용자 → 뷰어" 권한으로 공유되어 있어야** 정상 작동합니다.

가져오는 탭 6개 (gid로 고정 참조):
| 탭 | gid | 용도 |
|---|---|---|
| 피키 로우데이터(자동) | `0` | 크리에이터, 영상 링크, GMV, 조회수 등 (한 행 = 영상 1개) |
| 누리 로우데이터(자동) | `530164290` | 위와 동일한 구조 (영상 게시일 컬럼 있음) |
| GCD 로우데이터(수기) | `670984473` | 한 행 = 크리에이터 1명, 링크 여러 개를 옆으로 나열한 비정형 시트 |
| GEC 로우데이터(수기) | `704602447` | 누리와 동일한 구조 |
| 스파크애즈(수기) | `1393881713` | 계정/광고비/노출/클릭 (일별) |
| GMVMAX(수기) | `327690174` | Post ID 기준 영상별 광고 성과 (일별) |

시트 안에 새로운 탭을 추가로 만들거나 탭 이름을 바꿔도 상관없습니다 — 오직 URL의 `gid` 숫자로 찾기 때문에, gid만 바뀌지 않으면 계속 정상 작동합니다.

## 영상 리스트 통합 로직 (GEC 기준)

**GEC 로우데이터가 전체 영상의 기준(마스터) 리스트입니다.** 나머지 3개 시트(피키/누리/GCD)는 "이 영상이 어느 채널에서 모집됐는지"를 알려주는 출처 태그로만 사용합니다.

1. GEC에 있는 모든 영상을 기본으로 깔고, 같은 영상 링크(숫자 ID)가 피키/누리/GCD 중 어디에도 있으면 그 이름으로 **출처 배지**를 붙입니다 (우선순위: 피키 > 누리 > GCD > 어디에도 없으면 GEC).
2. 혹시 GEC에는 없는데 피키/누리/GCD에만 있는 영상이 있다면, 놓치지 않도록 함께 표시하되 **"GEC 미등록"** 배지로 구분해서 보여줍니다. (상단 안내 배너에도 개수가 뜹니다)
3. GCD 시트 안의 캠페인 세부명(예: "직접 KOL", "Glow Duo Defenders 1", "6월 GMV Challenge")은 구분하지 않고 전부 **"GCD"** 한 가지로 태그합니다.
4. GCD 링크의 숫자 ID 뒤에 붙는 `?`나 추가 파라미터는 자동으로 무시됩니다 (숫자만 정확히 추출).
5. GCD/피키 등에 크리에이터 이름 옆에 `(L0)`, `(L2)` 같은 티어 표기가 있으면 자동으로 읽어서 **티어 입력창의 기본값으로 제안**합니다 (수동으로 덮어쓰면 그 값이 우선합니다). 해당 영상이 GEC 중복으로 걸러지더라도 티어 힌트는 그대로 반영됩니다.
6. 영상 링크는 클릭하면 실제 틱톡 영상으로 바로 이동합니다.
